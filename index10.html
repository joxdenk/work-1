CSS in Details

  • CSS is a cascading style sheet used for styling HTML elements
  • CSS is responsible for the nice looking of the website

    • Styling HTML involves using selectors such as Id, class or element selectors.

CSS selectors

  1. ID selectors.These selectors used # for identifying the element in an HTML page. the used an id attribute to select a specific unique element on HTML
  2. Class selectors.These selectors used dots and a class name to identify the element in an HTml page. Class selectors used a class attribute to an element in an HTML file.
  3. Element SelectorsThese types of selectors used the element name to style the HTML elements. e.g body, head
  1. Inline CSS.
    Inline selectors are used for styling a particular element within the HTML by giving an element a style sign.
  2. Internal CSS
    This type of selector is used inside the HTML to style different elements within the HTML
  3. External Selectors.

    4. This type are selectors are used for styling diference elemements or a body outside HTML. A designer may give an element a class, id or use the attribute within the tag.
